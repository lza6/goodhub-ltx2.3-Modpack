# 🏗️ 系统架构

本文提供 LTX2.3_v4.0 的系统架构概览。

---

## 整体架构

```
┌─────────────────────────────────────────────┐
│             用户界面层 (Web UI)              │
│  Gradio + JavaScript (预设面板等扩展)        │
├─────────────────────────────────────────────┤
│           LTX2.3 核心层 (自定义节点)          │
│  LTXVideo 节点 (Python)                      │
├─────────────────────────────────────────────┤
│            ComfyUI 框架层                     │
│  节点图引擎 + 模型加载器 + 工作流管理         │
├─────────────────────────────────────────────┤
│           PyTorch + 模型层                   │
│  LTX2.3 22b 模型 + LoRA 适配器               │
└─────────────────────────────────────────────┘
```

---

## 关键组件

| 组件 | 说明 | 位置 |
|------|------|------|
| `app.pyd` | 主程序入口，包含激活逻辑 | 项目根目录 |
| `core.pyd` | 核心推理引擎 | 项目根目录 |
| `LTX2_3_*.pyd` | LTXVideo 节点实现 | 项目根目录 |
| `ComfyUI/` | ComfyUI 框架 | 子目录 |
| `tools/` | 改进工具脚本 | 子目录 |

---

## 数据流

```
用户操作 → Gradio 事件 → 节点参数 → 工作流 JSON → 推理执行 → 输出视频
```
