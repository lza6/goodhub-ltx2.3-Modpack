# 🛠️ 开发者手册

---

## 环境搭建

1. 安装 Python 3.10 或 3.11
2. 克隆仓库到本地
3. 安装依赖（如果有）

```bash
pip install -r requirements.txt
```

---

## 代码规范

- Python: PEP 8, black 格式化
- JavaScript: Prettier, 2 空格缩进
- Commit message: Conventional Commits

---

## 贡献流程

1. Fork 本仓库
2. 创建功能分支 `git checkout -b feature/xxx`
3. 编写代码 + 测试
4. 提交 `git commit -m "feat: xxx"`
5. 推送并创建 PR 到 `dev` 分支
