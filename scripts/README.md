# 用户脚本

本目录存放用户常用的快捷启动脚本。

---

## 📜 脚本列表

| 脚本 | 说明 |
|------|------|
| `launch.bat` | 快速启动 ComfyUI Web 界面 |
| `quick_activate.bat` | 一键激活 + 启动（计划中） |

---

## 🚀 使用方法

### 启动 ComfyUI

```batch
cd D:\LTX2.3_v4.0
scripts\launch.bat
```

或直接双击 `scripts\launch.bat` 文件。
